# Introduction

Welcome to the Physical AI & Humanoid Robotics textbook. This comprehensive resource covers the cutting-edge intersection of artificial intelligence and robotics, preparing you for the future of autonomous humanoid systems.

## Course Overview

This textbook is organized into 5 comprehensive modules, each containing 2 focused topics:

- **Module 1**: The Robotic Nervous System (ROS 2)
- **Module 2**: The Digital Twin (Gazebo & Unity)
- **Module 3**: The AI-Robot Brain (NVIDIA Isaac)
- **Module 4**: Vision-Language-Action (VLA)
- **Module 5**: Capstone Autonomous Humanoid Project

## Interactive Learning

This AI-Native textbook features an integrated RAG (Retrieval-Augmented Generation) chatbot that allows you to ask questions about the content in real-time. Simply select text and click the chatbot icon to get AI-powered explanations and clarifications.

## Hardware Integration

Throughout the textbook, you'll find practical notes about:
- RTX-enabled workstations for AI/robotics computation
- Jetson Orin Nano edge kits for embedded AI
- RealSense sensors for computer vision
- Cloud alternatives (AWS RoboMaker, NVIDIA Omniverse Cloud)
- Latency management strategies

## Getting Started

Begin with Module 1 to establish a solid foundation in robotic systems, then progress through each module sequentially or jump to topics that interest you most.
